# Tool Care Team Task Breakout

**Event:** Happy Farm Spring Opening
**Date:** February 28, 2026
**Team Size:** 4 people

---

## Overview

This team organizes, cleans, and labels tools and equipment across three storage areas.

---

## Storage Areas

1. **Tool/Storage Rack** - main hand tools, farm supplies and equipment
2. **Tool Lock Box** - secured items (for our more expensive tools)
3. **Under Garden Table** - overflow and larger equipment

---

## Tasks

### Organizing

- Group like tools together (all shovels in one area, all hoes together, etc.)
- Arrange by tool type and size
- Ensure frequently used tools are easily accessible

### Cleaning

- Remove dirt, mud, and debris from all tools
- Wipe down handles
- Clean metal surfaces

### Labeling

- Label tools for identification
- Labeling options available:
  - Sharpie marker (quick labels)
  - Dremel-style etcher (permanent on metal)
  - Wood burner (permanent on wooden handles)

### Garden Table Support

- Install wooden supports under garden table to prevent sinking into ground
- Check table stability after installation

---

## Coordination Notes

**Mulch Team:** Any tools found during west side cleanup or general farm cleanup should come to the Tool Care Team for organizing and cleaning.

**All Teams:** At end of day, return all tools to designated storage areas so Tool Care Team can ensure everything is in place.
